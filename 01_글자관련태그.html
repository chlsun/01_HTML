p(Paragraph, 문단, 단락) 태그

글을 작성하는 용도의 태그

pre 태그 처럼 형식이 유지되진 않음

글의 주제가 바뀔 떄 마다 p태그를 새로 작성

문단(p 태그) 사이에 여백이 생성됨

글자에 적용되는 태그

b(bold) : 단순히 글자를 굵게 표시하는 태그
i(intalic) : 단순히 글자를 기울여 표시하는 태그
strong 태그 : 글자(단어)를 강조하는 용도의 태그
em 태그 : 문장의 맥락을 변화시키는 태그(강조) 
  mark 태그: 형광팬 효과를 주는 태그

  u(under) 태그 : 밑줄 긋는 태그

  s 태그 : 글자 가운데 선을 긋는 태그(취소, 오타)

  del : 삭제선을 긋는 태그(있다가 없어짐)

  * s, del 용도에 맞게 사용!
  걸그룹 멤버 명단 : 미영(탈퇴), 숙희, 영희(오타), 영숙

  small 태그 : 글자를 작게 표시하는 태그

  sup 태그(윗첨자) : x3

  sub 태그(아랫첨자) : log2

  abbr(abbreviation : 약어) 태그 : 마우스 오버 시 툴팁 제공
